Introduction

JavaScript is a versatile programming language commonly used in web development. It was created in 1995 by Brendan Eich while he was working at Netscape Communications Corporation. Initially developed in just 10 days, JavaScript has since become one of the core technologies of the World Wide Web, alongside HTML and CSS.

Timeline

  • 1995: JavaScript was created by Brendan Eich.
  • 1996: Microsoft introduced JScript with Internet Explorer 3.0.
  • 1997: ECMAScript 1 was released.
  • 1999: ECMAScript 3 was released, becoming widely adopted.
  • 2009: ECMAScript 5 was released, adding many features.
  • 2015: ECMAScript 6 (ES6) was released, introducing significant updates.
  • 2021: ECMAScript 12 (ES12) was released.

Key Features

Over the years, JavaScript has evolved to include many powerful features:

  • Dynamic Typing
  • First-class Functions
  • Prototypes
  • Async/Await
  • Modules
  • Event-driven programming

Fun Facts

  • JavaScript was originally named Mocha, then renamed to LiveScript, and finally to JavaScript.
  • Despite its name, JavaScript is not related to Java.
  • JavaScript can be used both on the client-side and server-side (e.g., Node.js).
